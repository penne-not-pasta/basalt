# Basalt Cloud Storage

Basalt is a Software that can Implement multiple Cloud Providers Into one App for easy Access to your Cloud Storage, we also hope to add SFTP Support in the future for accessing Your Cloud Servers Files Mre Efficiently.

# Support For Cloud Providers
Basalt Currently supports only **Google Drive** since this project is just starting but we are going to add more very soon
